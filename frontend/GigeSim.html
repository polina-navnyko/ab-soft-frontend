GigESim

GigESim is a component software solution that allows end-user and developers to turn their computer into a virtual GigE Vision camera. Any GigE Vision compliant software running at the same or other computers on the network will treat GigESim as an actual camera with fully adjustable features. The package includes a powerful and easy-to-use GigE Server SDK that lets your applications stream images to multiple computers on the network and provides a full control over your application from remote locations.

With GigESim you can:

  • Make your computer behave like a camera fully compatible with the GigE Vision and GenICam standards.
  • Create one or several virtual camera objects on one computer host.
  • Stream individual images or video sequences to the network and receive it at a remote node using standard GigE Vision software.
  • Extend your throughput by using several network connections in parallel.
  • Transmit the video from each of your virtual cameras to multiple computers in the network in the multicast mode.
  • Assign GenICam-compatible features to your virtual camera and control your application from remote computers.
  • Perform the development and testing of a client GigE Vision application when a real camera is not available.
  • Select generated patterns or pre-recorded AVI files as a streaming video source.
  • Split image data to multiple streams.
  • Convert cameras of standard types (analog, USB, 1394, CameraLink) into simulated GigE Vision cameras.
  • Simulate dual-port GigE Vision cameras and network switches.
  • Convert an RGB color video to Bayer raw video to reduce the image payload size.
  • Implement distributed image processing on several computers in the parallel or/and pipeline modes.
  • Implement a customized remote control over your application.
  • Prototype development and design of a GigE Vision camera .
  • Simulate rigorous network conditions by disordering and skipping video packets.
GigESim is an easy-to-use and flexible solution for software testing, distributing computing, application remote control, multi-node surveillance, pipeline video transfer and many more. GigESim can be used immediately, without adaptation. All you need is a Gigabit Ethernet connection between your computers!.
